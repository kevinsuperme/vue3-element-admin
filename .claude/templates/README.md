## 📝 模板规范

### 🎯 命名规范
```yaml
模板文件:
  - 使用描述性名称
  - 包含文件扩展名
  - 小写字母 + 连字符

占位符:
  - 使用描述性名称
  - 大写字母开头
  - 语义化命名

示例:
  - ComponentName: 组件类名
  - componentName: 组件名(kebab-case)
  - ApiModule: API模块名
  - StoreModule: Store模块名
```

### 🔧 结构规范
```yaml
文件结构:
  1. 模板注释 (使用说明)
  2. 基础结构
  3. 占位符标记
  4. 实现示例

代码规范:
  - 遵循项目编码规范
  - 包含必要注释
  - 提供使用示例
  - 标记可替换部分
```
